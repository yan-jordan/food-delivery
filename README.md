## 🚀 Features

- Fully responsive design
- Dynamic food menu with images and pricing
- Add-to-cart functionality using Context API
- Reusable React components
- Smooth layout and styling
- Mobile friendly design

---

## 🛠️ Technologies Used

- React.js (with Hooks)
- JavaScript (ES6+)
- CSS3
- React Context API
- Vite (for development setup)

## 📦 Getting Started

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/your-username/tomato-food-delivery.git

# Navigate into the project folder
cd tomato-food-delivery

# Install dependencies
npm install

# Start the development server
npm run dev
